Proyecto: Biblioteca SPA - Módulo 6

Aplicación de gestión de datos desarrollada con Vue.js, diseñada bajo el patrón MVVM para ofrecer una experiencia de usuario fluida como Single Page Application (SPA).

● Resumen del Proyecto
Este proyecto implementa una biblioteca digital que permite gestionar y visualizar libros mediante una navegación fluida, utilizando Vue Router para la transición entre vistas sin recarga de página.

 ● Decisiones Técnicas y Estructura
Para cumplir con los estándares de desarrollo solicitados, se implementaron las siguientes soluciones:
- Estructura de Componentes: Aplicación de modularidad mediante componentes reutilizables (como LibroItem.vue), facilitando la escalabilidad del código.
- Reactividad y Binding: Uso de v-model para la sincronización bidireccional en formularios dinámicos.
- Directivas: Aplicación efectiva de v-if para renderizado condicional, v-for para el despliegue de listados de libros y v-show para control de visibilidad.
- Gestión de Estado: Centralización de la información en store.js, asegurando la consistencia de datos bajo el patrón MVVM.
- Navegación: Configuración de rutas dinámicas mediante Vue Router para una navegación SPA fluida.
- Eventos: Uso de métodos y modificadores de eventos (@click, entre otros) para capturar la interacción del usuario.

● Cómo ejecutar el proyecto
1.	Clonar el repositorio: git clone https://github.com/denisevasquez24dv-dotcom/proyecto-biblioteca-spa.git
2.	Instalar dependencias: npm install
3.	Iniciar el servidor de desarrollo: npm run serve
